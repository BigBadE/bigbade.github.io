My Contributions/Projects

Cobertura

Cobertura is a tool to calculate test coverage percentages and verify Java code. It is used by a few popular tools, most notably including CodeCov. Sadly, it was majorly out of date and didn't even verify Java 8 code correctly. I learned a source-code generating library called JavaNCSS and updated it to the latest version. PR #407


Clairvoyant

Clairvoyant is a WIP Battlesnake that implements Minimax with alpha-beta pruning. It's a heavy work in progress, as it's a difficult thing to implement. BigBadE/Clairvoyant


JavaSkript

JavaSkript is a project meant to compile English text (written in the Skript standard) into Java. Still also a WIP, it has a fully functioning system to programmatically write a class file. BigBadE/JavaSkript
